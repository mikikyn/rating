# 📊 Система прогнозирования академической успеваемости

## 📌 Описание проекта

Данный проект представляет собой простую систему машинного обучения, которая предсказывает оценку студента на основе количества часов обучения.

Модель использует метод **линейной регрессии**, чтобы найти зависимость между:

* временем обучения (study_hours)
* итоговой оценкой (grade)

В проект входят:

* обучение модели
* скрипт для предсказания
* веб-приложение (Streamlit)

---

## ⚙️ Используемые технологии

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Joblib
* Streamlit

---

## 📁 Структура проекта

```
project/
│
├── dataset_study.csv      # Датасет
├── train.py               # Обучение модели
├── predict.py             # Предсказание в консоли
├── main.py                # Веб-приложение (Streamlit)
├── study_model.v1         # Сохранённая модель (создаётся после обучения)
```

---

## 🧠 Как работает система

1. Загружается датасет из CSV файла
2. Модель обучается с помощью линейной регрессии
3. Обученная модель сохраняется в файл
4. Модель используется для предсказания оценки

---

## 🚀 Как запустить проект

### 1. Установить зависимости

```
pip install pandas scikit-learn matplotlib joblib streamlit
```

---

### 2. Обучить модель

```
python train.py
```

После этого создастся файл:

```
study_model.v1
```

---

### 3. Запуск предсказания (консоль)

```
python predict.py
```

---

### 4. Запуск веб-приложения

```
streamlit run main.py
```

---

## 📊 Входные и выходные данные

* Входной параметр: `study_hours` (часы обучения)
* Выходной параметр: `grade` (оценка)

---

## 📈 Модель

Используется метод **линейной регрессии**.

Модель строит зависимость вида:

```
grade = a * study_hours + b
```

---

## 💡 Пример

Если студент учится:

```
5 часов
```

Модель может предсказать:

```
примерно 70–80 баллов
```
